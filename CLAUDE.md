# Working preferences for this repo

## Git

- Commit whenever a change is verified working (tests pass, type-check clean, and/or live-verified) — don't wait to be asked. This overrides the general "only commit when explicitly asked" default for this repo specifically.
- `git push` still needs its own confirmation each time — this instruction covers committing, not pushing.
